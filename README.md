# Companydle (S&P 500 guessing game)

A fully static Wordle-style guessing game inspired by Royaledle (attribute matching),
but for S&P 500 companies.

Daily reset: **midnight New York time** (America/New_York).

## Run locally
Just open `index.html` in a browser.

## Deploy on GitHub Pages
1. Push these files to a GitHub repo
2. Repo Settings → Pages → Deploy from Branch → `main` + `/root`
